# 启动说明

当前项目已经升级为 FastAPI + React + PostgreSQL/pgvector + Redis 的全栈多 Agent 项目。

完整启动、密钥配置、Docker 部署和 GitHub 上传说明都在：

```text
README.md
```

最短启动方式：

```powershell
Copy-Item server/.env.example server/.env
docker compose up --build
```

访问：

```text
http://127.0.0.1:5173
```

注意：真实 API Key 只写入 `server/.env`，不要提交 `.env` 文件。
